# [Free Design Services](https://planetpeanut.studi/services)

Hello! I'm [Hylke](https://planetpeanut.studio). I provide free design services for ***Free and Open Source*** projects.

![Banner](.github/services.png)

Made possible by [community sponsors](https://planetpeanut.studio/sponsors). You don't need to be a sponsor to request a service, but even a [$1 / month](https://planetpeanut.studio/sponsors) sponsorship does add up!

Your project is elligible when it:

* uses an OSI approved <a href="https://opensource.org/licenses">license</a>
* is published on <a href="">Flathub</a>, or will be
* is not vibe-coded slop or crypto/gambling

<br>

## 1 ～  [App Icons](https://planetpeanut.studio/services#app-icons)

## 2 ～ [Symbolic Icon Sets](https://planetpeanut.studio/services#symbolic-icon-sets)

## 3 ～ [Flathub Reviews](https://planetpeanut.studio/services#flathub-reviews)

I'll review and edit your app's MetaInfo file and help you pass Flathub's <a href="https://docs.flathub.org/docs/for-app-authors/metainfo-guidelines/quality-guidelines">Quality Guidelines</a>.

[Details](https://planetpeanut.studio/services#flathub-reviews) | [Request for free →](https://github.com/hbons/free-services/issues/new?template=TEST.yml)
